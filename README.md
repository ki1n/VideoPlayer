# VideoPlayer
## Архитектура приложения

### Структура приложения

Приложение разбито на 3 слоя: data, domain, presentation

### Стэк технологий

* Clean Architecture, MVVM
* Kotlin
* Android X: ViewBinding, RecyclerView, ConstraintLayout, Lifecycle
* DI: Dagger
* Threading: RxJava2 (RxAndroid)
* Network: Retrofit, GSON
* Navigation: Cicerone
* UI: Google Material, Adapter Delegates
* Images: Glide
* Logging: Timber

Тестов пока нет.

## Стиль проекта

- Стиль кодирования на Kotlin: https://developer.android.com/kotlin/style-guide
